# API

`Inkar Suki`官方文档开放了极少数接口供`Inkar Suki`实例调用。

本站`Token`请自行在互联网搜索，**请勿提交`JX3API`的`token`！**

如果没有特殊说明，所有参数均为必选！

## 交易行物品

> 贡献者：`HornCopper`。

> 请求方式(GET)

```
https://inkar-suki.codethink.cn/trade
```

> 请求参数

```json
{
    "server": "幽月轮",
    "name": "泣麟砂",
    "token": "114514"
}
```
